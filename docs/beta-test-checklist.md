## テストの進め方

1. 各セクションを上から順にチェックしてください
2. 問題があったら「問題の内容」「どの画面で」「何をしたか」をメモしてください
3. サンプルデータが入っているので、自由に操作して大丈夫です
4. データを壊しても設定 > サンプルデータ投入で復元できます

## フィードバックの記入

テスト完了後、以下の観点でフィードバックをお願いします：

1. **使いにくかった画面・操作**（どこで何が分かりにくかったか）
2. **足りないと思った機能**（日常業務で必要だが見当たらないもの）
3. **データの表示がおかしい箇所**（数値・表示・計算結果の不一致）
4. **表示速度が遅い画面**（体感で3秒以上かかるもの）
5. **その他気づいたこと**
